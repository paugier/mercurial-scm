About this website

This website is a project of the Mercurial community. Its design, initial Django and hosting was provided by bitbucket.org. Content was provided by Arne Babenhauserheide. The templates were written by Steve Losh. Additional coding, content and conversion by David Soria Parra. The website is using flask.

The repository can be found at https://www.mercurial-scm.org/repo/hg-website. You are welcome to clone and improve the website.

Help us

This website always needs improvement. Please send any kind of patches, suggestions and comments to either the Mercurial mailinglist or directly to us.

People often ask what exactly can be done to improve the website. First of all, check our ongoing todo list. Second, read the pages and fix our poor grammar. Third, write guides, write about extensions and contribute to the Wiki. If you'd like to help with additional graphics or something completely different (we have wishlists), you are welcome.
