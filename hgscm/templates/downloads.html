Get started

Mercurial is written in python with platform independence in mind. As a result, Mercurial is available on Microsoft Windows, GNU/Linux, MacOS X, OpenSolaris and others. You can either download a binary package for the system of your choice or build if from sources.

Packages for common Linux, BSD and Solaris distributions can be installed from the system specific repositories

# Debian/ubuntu
$ apt-get install mercurial

# Fedora
$ yum install mercurial

# Gentoo
$ emerge mercurial

# OpenSolaris
$ pkg install SUNWmercurial
